MatrixIT Mobile TCP Client

Учебное Android-приложение на .NET MAUI для подключения к TCP-серверу, отправки и получения текстовых данных.

## Возможности

- подключение по IP-адресу и порту;
- отправка UTF-8 сообщений;
- асинхронный приём данных;
- ручное отключение;
- тайм-аут подключения;
- проверка IP-адреса и порта;
- локальная история сообщений в SQLite;
- очистка истории;

## TCP-протокол
Исходящие сообщения кодируются в UTF-8.
В конец отправленного текста добавляется символ `\\n`.
Входящие данные отображаются блоками по мере их получения из TCP-потока.

## Запуск
1. Открыть решение в Visual Studio
2. Установить workload .NET MAUI
3. Запустить TCP-сервер
4. Выбрать Android Emulator или физическое устройство
5. Ввести IP-адрес и порт сервера
6. Нажать «Подключиться»

Для сервера на компьютере и стандартного Android Emulator используется адрес 10.0.2.2

- LogMessage — модель сообщения
- DatabaseService — работа с SQLite
- TerminalViewModel — состояние экрана, TCP и команды
- MainPage.xaml — пользовательский интерфейс
- MainPage.xaml.cs — только инициализация страницы
